# Toast Notification
Project #27 in Brad Traversy's 50 Projects in 50 Days. In this project, a notification button is created which is used to notify a user of an error and this notification pops-up and then disappears after a few seconds. This can be created in BootStrap.

![Toast Notification](screenshot.jpg)

## Built With
- [HTML5](https://developer.mozilla.org/en-US/docs/Web/Guide/HTML/HTML5)
- [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS)
- [Javascript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
